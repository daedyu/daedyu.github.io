## DML 이란
dml은 데이터 조작어로 실제로 table 안 데이터를 조작할 수 있는 언어다.
> 삽입, 수정, 삭제, 조회가 가능하다.
{.prompt-info}

## SQL DML 문법
sql에서 dml 문법은
- insert
- select
- update
- delete
등이 있다.

## insert
테이블에 데이터를 삽입하는 언어다.

코드는 다음과 같다.
```sql
insert into 테이블명 [(속성들)]
values
(넣을 값)
```

> 예시문제
> 
> 고객 테이블에 ('김민규', 30포인트)를 추가한다.
> 
---

```sql
insert into customer (name, )
```
